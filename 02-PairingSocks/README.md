# Problem Statement

There is a large pill of socks that must be paired. The colors of the socks will be represented by characters. Determine how many pairs of socks with matching colors there are.

The arr input is:

['a', 'b', 'c', 'd', 'e', 'a', 'g', 'a']

The output should be:

---

1
